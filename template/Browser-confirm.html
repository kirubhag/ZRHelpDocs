Browser Confirm Box

In this page you can have quick overview and some features of Browser Confirm Modal Box component.

About Browser Confirm Box

It is also similar to Confirmation box,except you can't change it's default functionalities like can't change the button caption, It's alignment at all and you can't change it's title and content. 

The ZRComponent.browserConfirmDialog() method displays a default confirm box that is similar to browser confirm box.
