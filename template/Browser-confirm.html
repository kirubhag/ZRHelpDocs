Here's a quick overview of the main features of Browser Confirm Modal Box component.

About Browser Confirm Box

This is similar to the Confirmation box, except you can't change the default functionalities like can't change the button caption, It's alignment at all and you can't change it's title and content. 

The ZRComponent.browserConfirmDialog() method displays a default confirm box that is similar to browser confirm box.
