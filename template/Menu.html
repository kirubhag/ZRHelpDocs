About Confirm Box

A confirm box is often used if you want the user to verify or accept something. 

The ZRComponent.createConfirmDialog() method displays a dialog box with a specified message, along with an OK and a Cancel button.

You can customize this confirmation modal box by providing the values as JS object.

title

The title attribute is used to set the title of the confirm modal box.

ZRComponent.createConfirmDialog({title:"My Custom Modal Box"});
content

The content attribute is used to set the content of the confirm modal box.

ZRComponent.createConfirmDialog({content:"This is Custom Modal Box"});
btnAlignClass

The btnAlignClass attribute is used to align the button control of the confirm modal box according to the user input.Button allowed alignment classes are "al, ac, ar".If you are giving irrelevent class name rather than in the specified list it will be ignored automatically.

ZRComponent.createConfirmDialog({btnAlignClass:"al"});
modalClass

The modalClass attribute is used to increase or decrease the with of the modal box. Allowed width values are "wid400, wid600, wid700, wid800".If you are giving irrelevant class name rather than in the specified list it will be ignored automatically.

ZRComponent.createConfirmDialog({modalClass:"wid400"});
position

The position attribute is used to place the modal box at the specified area. Allowed values are "top", "middle", "bottom", "middletop", "middlebottom".If you are giving irrelevant class name rather than in the specified list it will be ignored automatically.

ZRComponent.createConfirmDialog({position:"middle"});
removeFreeze

By default modal box has freeze layer.But it can be removed by the attribute called removeFreeze. The removeFreeze attribute ensure that either freeze layer need to appear or not.It accept boolean value only. If the removeFreeze value is true then it will remove the freeze layer.

ZRComponent.createConfirmDialog({removeFreeze:true});
