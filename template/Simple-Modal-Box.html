Simple Modal Box

In this page you can have quick overview and some features of Simple Modal Box component.

About Browser Confirm Box

It's just a modal box.You can customize as your wish. You can make it as a simple HTML form.

The ZRComponent.openSimpleModalBox() method displays a default confirm box that is similar to browser confirm box.
To generate HTML form you can use the method  Utils.createHTML().Pass this as parameter to modalContent object.
modalContent

The modalContent attribute is used to set the content or any HTMl string into to Modal.

ZRComponent.openSimpleModalBox( {
    modalContent : '
        <div class="overview-head">
        <h3 class="m0">
        About Browser Confirm Box</h3
        </div>'
} );
var input = Utils.createHTML({
   "name": "input",//NO I18N
   "attr": { //NO I18N
   "type": "button",//NO I18N
   "class": "btn-primary",//NO I18N
   "value": "Save" //NO I18N
   }
});

ZRComponent.openSimpleModalBox( {
content : input
} );
